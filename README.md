# 📚 AI Study Assistant

An AI-powered study assistant built using **FastAPI + Groq + HTML/CSS/JavaScript**.

---

## 🚀 Features

- ✨ Summarize study notes
- ❓ Generate quizzes (MCQs)
- 🧠 Explain topics simply
- 📄 Upload PDF and extract summary
- 🃏 Generate flashcards
- 💬 Ask AI anything
- 📋 Generate forms automatically

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI (Python)
- AI Model: Groq (LLaMA 3)
- PDF Handling: PyPDF2
